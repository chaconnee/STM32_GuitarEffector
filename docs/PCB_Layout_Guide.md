# WM8978 音频板 PCB 布局指南

## 板子参数

| 参数 | 值 |
|---|---|
| 层数 | 2 层 |
| 尺寸 | 待定（建议 50x40mm 左右） |
| 板厚 | 1.6mm |
| 铜厚 | 1oz |

## 一、元件布局（Placement）

### 布局原则

```
┌─────────────────────────────────────────────┐
│  USB                                    OUT  │
│  ┌───┐    ┌─────┐                     ┌──┐ │
│  │USB├──→ │ LDO ├──→ 3V3_A            │  │ │
│  └───┘    │ LDO ├──→ 3V3_D        ┌───┤  │ │
│     │     └─────┘                  │WM │  │ │
│     │         ┌──────┐             │89 │TRS│ │
│     │         │ 排针  │←──I2S/I2C──│78 │  │ │
│     │         │(接MCU)│            │   │  │ │
│     │         └──────┘             └───┴──┘ │
│     └── TVS ──→ 5V                          │
└─────────────────────────────────────────────┘
```

### 分区规则

| 区域 | 位置 | 包含元件 |
|---|---|---|
| **电源区** | 左上 | USB、TVS、LDO、滤波电容 |
| **数字接口区** | 中间 | 排针、I2S/I2C 上拉电阻 |
| **模拟区** | 右侧 | WM8978、模拟去耦电容、MICBIAS |
| **输出区** | 右下 | TRS 连接器、隔直电容、保护电阻 |

### 关键元件放置顺序

1. **WM8978** — 放在板子中心偏右，靠近输出连接器
2. **排针（接 MCU）** — 放在 WM8978 左侧，I2S/I2C 走线最短
3. **LDO** — 靠近电源输入，远离音频路径
4. **TRS 连接器** — 板子边缘，方便接线
5. **USB 连接器** — 板子边缘

## 二、地平面分割（Ground Split）

### 分割示意

```
┌─────────────────────────────────────────────┐
│                                             │
│   ┌─────────────┐    ┌─────────────────┐   │
│   │             │    │                 │   │
│   │   DGND      │    │     AGND        │   │
│   │  (数字地)    │←──→│   (模拟地)      │   │
│   │             │单点│                 │   │
│   │  STM32      │连接│  WM8978         │   │
│   │  USB        │    │  TRS            │   │
│   │  LDO_D      │    │  LDO_A          │   │
│   │             │    │                 │   │
│   └─────────────┘    └─────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

### 单点连接位置

**在 WM8978 的 AGND 和 DGND 引脚之间单点连接：**

```
WM8978 引脚连接:
Pin 12 (DGND) ──→ DGND 平面
Pin 28 (AGND) ──→ AGND 平面
                 │
                 └── 两个平面在 WM8978 下方单点连接
```

### 二层板地平面策略

**顶层（Top Layer）：**
- 信号走线
- 元件焊盘
- 地铜皮填充（尽量多）

**底层（Bottom Layer）：**
- 完整地平面（分割为 DGND 和 AGND）
- 电源走线
- 少量信号走线

## 三、电源走线（Power Routing）

### 电源树

```
USB VBUS (5V)
    │
    ├──[D4 肖特基]──→ 5V (系统电源)
    │
    ├──[LDO1]──→ 3V3_A (模拟电源)
    │              │
    │              ├── AVDD (WM8978 Pin 31)
    │              ├── DBVDD (WM8978 Pin 14)
    │              └── VMID 去耦
    │
    └──[LDO2]──→ 3V3_D (数字电源)
                   │
                   ├── DCVDD (WM8978 Pin 13)
                   └── 排针输出（给外部 MCU 供电）
```

### 电源走线规则

| 电源 | 走线宽度 | 去耦电容位置 |
|---|---|---|
| 5V (VBUS) | ≥ 0.5mm | USB 口附近 |
| 3V3_A | ≥ 0.3mm | WM8978 AVDD 引脚旁 |
| 3V3_D | ≥ 0.3mm | STM32 VDD 引脚旁 |
| SPKVDD | ≥ 0.3mm | WM8978 SPKVDD 引脚旁 |

---

## 四、关键走线（Critical Routing）

### I2S 信号（排针 ↔ WM8978）

| 信号 | 说明 | 走线要求 |
|---|---|---|
| I2S2_SCK | 位时钟 | 短、直、包地 |
| I2S2_WS | 字时钟 | 短、直、包地 |
| I2S2_SD | 数据 | 短、直、包地 |
| I2S2_MCK | 主时钟 | 短、直、包地 |

```
排针 ←──── 短距离 ────→ WM8978
  │                        │
  └── I2S 走线 < 20mm ────┘
```

### I2C 信号（排针 ↔ WM8978）

| 信号 | 说明 | 走线要求 |
|---|---|---|
| I2C_SCL | 时钟 | 短、加上拉 |
| I2C_SDA | 数据 | 短、加上拉 |

### 音频输出（WM8978 → TRS）

| 信号 | 说明 | 走线要求 |
|---|---|---|
| LOUT1 | 左声道 | 粗线、远离数字信号 |
| ROUT1 | 右声道 | 粗线、远离数字信号 |

## 五、去耦电容放置

### WM8978 去耦

| 电容 | 位置 | 值 |
|---|---|---|
| AVDD 去耦 | Pin 31 旁边 | 100nF + 4.7µF |
| DCVDD 去耦 | Pin 13 旁边 | 100nF |
| DBVDD 去耦 | Pin 14 旁边 | 100nF |
| SPKVDD 去耦 | Pin 26 旁边 | 100nF + 4.7µF |
| VMID 去耦 | Pin 27 旁边 | 100nF + 4.7µF |

### 排针去耦（给外部 MCU 供电）

| 电容 | 位置 | 值 |
|---|---|---|
| 3V3_D 输出 | 排针附近 | 100nF |

## 七、检查清单

### 布局检查

- [ ] WM8978 散热焊盘连接到 AGND
- [ ] 排针靠近 WM8978，I2S/I2C 走线短
- [ ] USB TVS 靠近连接器
- [ ] LDO 靠近电源输入
- [ ] 音频输出远离数字信号

### 地平面检查

- [ ] AGND 和 DGND 在 WM8978 下方单点连接
- [ ] 底层地平面完整，无信号线割断
- [ ] 散热焊盘有多个过孔连接到地平面

### 电源检查

- [ ] 每个电源引脚有去耦电容
- [ ] 去耦电容紧靠 IC 引脚
- [ ] 电源走线宽度足够

### 信号检查

- [ ] I2S 走线短且直（< 20mm）
- [ ] I2C 上拉电阻靠近排针
- [ ] 音频输出走线加粗（≥ 0.3mm）
