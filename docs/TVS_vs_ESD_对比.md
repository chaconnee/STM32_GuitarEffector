# TVS 管 vs ESD 管 对比

> ESD 管是 TVS 管的子集

```
TVS 管 (瞬态抑制二极管)
  ├── 大功率 TVS（防雷击、浪涌）
  │     ├── SMBJ5.0A (通流 6A)
  │     ├── SMAJ5.0A (通流 5A)
  │     └── 1.5KE 系列 (通流 5A+)
  │
  └── ESD 保护二极管（防静电）★ 子集
        ├── USBLC6-2SC6
        ├── PRTR5V0U2X
        └── KLXES15AAA1
```

## 核心区别

| 特性 | TVS 管 | ESD 管 |
|---|---|---|
| 保护对象 | 浪涌、雷击、电源尖峰 | 静电放电 |
| 能量等级 | 高（焦耳级） | 低（微焦耳级） |
| 通流量 | 1A ~ 100A+ | 0.1A ~ 1A |
| 响应时间 | < 1ns | < 1ns |
| 结电容 | 较大（几十~几百 pF） | 极小（< 1pF） |
| 封装 | SMA/SMB/DO-214 | SOT-23/SOT-323 |
| 典型应用 | 电源线、天线、通信线 | USB、HDMI、音频、I2C |

## 选型指南

| 应用场景 | 推荐 | 原因 |
|---|---|---|
| 电源输入保护（5V/12V/24V） | TVS | 能量大，承受浪涌 |
| USB/HDMI 信号线 | ESD 管 | 电容小，不影响信号 |
| 天线、长线缆 | TVS | 能量大，防雷击 |
| I2C/SPI/按键 | ESD 管 | 体积小，电容小 |
| USB 供电（不传信号） | TVS | 不需要低电容 |

## 为什么 ESD 管电容小？

ESD 管专为高速信号设计：
- PN 结面积小 → 电容小 → 不影响信号完整性

TVS 管为了承受大电流：
- PN 结面积大 → 电容大 → 会破坏高频信号
