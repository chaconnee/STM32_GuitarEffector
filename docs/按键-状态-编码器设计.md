### 全局状态与数据定义

#### 状态机

系统使用有限状态机，共四个状态：

```c
typedef enum {
    ST_MAIN,   // 主页
    ST_AMP,    // 前级参数编辑
    ST_CAB,    // 箱体IR选择
    ST_RVB     // 混响参数编辑
} State_t;
```
```c
static State_t state = ST_MAIN;   // 当前状态
static uint8_t subpage = 0;       // 混响子页 (Decay/Mix/Tone)
static uint8_t chain_pos = 0;     // 主页高亮效果链位置 (0:AMP,1:CAB,2:RVB)
```

#### 保存的参数与全局变量

```c
static float amp_drive_val = 0.0f;
static float rv_decay_val  = 0.95f;
static float rv_mix_val    = 0.5f;
static float rv_tone_val   = 0.5f;
static uint8_t cab_ir_idx  = 0;
volatile float g_master_volume = 1.0f;
```

### 按键

#### Choose（PA1）—— 循环高亮效果链位置

仅在主页有效，每次按下顺序切换高亮目标（AMP → CAB → RVB），并触发 LED 快速闪烁 4 次（200ms 周期），以提示当前选中的效果。

使用上升沿检测函数 `Btn_Rise` 捕获按键动作；在 `ST_MAIN` 状态下递增 `chain_pos`，对 3 取模实现循环；利用 `main_blink` 计数器控制闪烁次数，每次翻转 LED 后递减；启动闪烁前先熄灭 LED，保证闪烁序列整齐。

```c
if (Btn_Rise(&pa1_last, pa1)) {
    if (state == ST_MAIN) {
        chain_pos = (chain_pos + 1) % 3;               // 循环 0→1→2→0
        main_blink = (chain_pos + 1) * 2;              // 闪烁次数: 2,4,6
        main_blink_timer = HAL_GetTick();
        led_on = 0;
        HAL_GPIO_WritePin(GPIOC, GPIO_PIN_13, GPIO_PIN_SET); // LED 熄灭
    }
}
```

#### Enter（PB0）—— 进入参数模式 / 内部切换

- **主页**：根据当前高亮效果进入对应的参数编辑模式（`ST_AMP` / `ST_CAB` / `ST_RVB`）。
- **AMP 模式**：无操作（防误触）。
- **CAB 模式**：循环切换箱体 IR，立即生效。
- **RVB 模式**：循环切换混响子页（Decay → Mix → Tone）。

```c
if (Btn_Rise(&pb0_last, pb0)) {
    switch (state) {
    case ST_MAIN:
        if (chain_pos == 0)      EnterState(ST_AMP, 0);
        else if (chain_pos == 1) EnterState(ST_CAB, 0);
        else                     EnterState(ST_RVB, 0);
        break;
    case ST_AMP:
        break;   // 无动作
    case ST_CAB:
        cab_ir_idx = (cab_ir_idx + 1) % IR_COUNT;
        CabSim_SelectIR(cab_ir_idx);
        break;
    case ST_RVB:
        subpage = (subpage + 1) % 3;   // 0→1→2→0
        break;
    }
}
```

##### 状态进入函数

```c
static void EnterState(State_t s, uint8_t sub) {
    state = s;
    subpage = sub;
}
```

#### Bypass（PB1）—— 旁通切换

在任意状态下均可切换当前作用效果的旁通状态。

- **主页**：根据高亮位置 `chain_pos` 切换对应效果的旁通。
- **参数模式**：直接切换该模式代表的效果（AMP / CAB / RVB）的旁通。

```c
if (Btn_Rise(&pb1_last, pb1)) {
    if (state == ST_MAIN) {
        if (chain_pos == 0)      amp_sim_effect.bypassed = !amp_sim_effect.bypassed;
        else if (chain_pos == 1) cab_sim_effect.bypassed = !cab_sim_effect.bypassed;
        else                     reverb_effect.bypassed  = !reverb_effect.bypassed;
    } else if (state == ST_AMP) {
        amp_sim_effect.bypassed  = !amp_sim_effect.bypassed;
    } else if (state == ST_CAB) {
        cab_sim_effect.bypassed  = !cab_sim_effect.bypassed;
    } else if (state == ST_RVB) {
        reverb_effect.bypassed   = !reverb_effect.bypassed;
    }
}
```

#### Exit（PA7）—— 保存并返回主页

仅在非主页状态有效，保存当前参数并返回 `ST_MAIN`。主页下无动作。

调用 `ExitState` 将临时参数值写入对应效果器（确保编辑后的参数生效），然后状态置为主页。AMP 和 RVB 的参数分别通过 `set_param` 推送到 DSP 效果模块。

```c
static void ExitState(void) {
    amp_sim_effect.set_param(&amp_sim_effect, 0, amp_drive_val);
    reverb_effect.set_param(&reverb_effect, 0, rv_decay_val);
    reverb_effect.set_param(&reverb_effect, 1, rv_mix_val);
    reverb_effect.set_param(&reverb_effect, 2, rv_tone_val);
    state = ST_MAIN;
}
```
```c
if (Btn_Rise(&pa7_last, pa7)) {
    if (state != ST_MAIN) ExitState();
}
```

### 辅助逻辑实现

#### 按键去抖与上升沿检测

所有按钮均通过一个统一的去抖机制和上升沿检测函数处理，避免抖动误触发。
```c
static bool Btn_Rise(GPIO_PinState *last, GPIO_PinState now) {
    if (*last == GPIO_PIN_RESET && now == GPIO_PIN_SET) {
        *last = now;
        return true;
    }
    *last = now;
    return false;
}
```
    主轮询中每 100ms 才读取一次按键，保证去抖稳定：
```c
if (HAL_GetTick() - debounce < 100) return;
debounce = HAL_GetTick();
// ... 读取 GPIO ...
```

#### 编码器调节与参数映射

编码器增量 `enc_delta` 乘以步长 `0.005f`，根据当前状态/子页更新不同参数。主页和 CAB 模式下统一调整主音量，其他模式更新各自参数并实时写入效果器：
```c
float step = (float)enc_delta * 0.005f;
switch (state) {
case ST_MAIN:
case ST_CAB:
    g_master_volume = CLAMP(g_master_volume + step, 0.0f, 1.0f);
    break;
case ST_AMP:
    amp_drive_val = CLAMP(amp_drive_val + step, 0.0f, 1.0f);
    amp_sim_effect.set_param(&amp_sim_effect, 0, amp_drive_val);
    break;
case ST_RVB:
    if (subpage == 0) { rv_decay_val = CLAMP(...); reverb_effect.set_param(...); }
    if (subpage == 1) { rv_mix_val   = CLAMP(...); reverb_effect.set_param(...); }
    if (subpage == 2) { rv_tone_val  = CLAMP(...); reverb_effect.set_param(...); }
    break;
}
```

#### LED 状态指示

`LED_Update` 函数根据当前状态和闪烁需求控制板载 LED：

- **主页无闪烁**：任一效果未旁通则 LED 常亮，否则熄灭。
- **主页闪烁中**：以 200ms 周期翻转 LED，直至 `main_blink` 减为 0。
- **参数模式**：按不同频率闪烁（AMP: 1s 周期, CAB: 600ms, RVB: 350ms），以区分模式。

```c
static void LED_Update(void) {
    uint32_t now = HAL_GetTick();
    if (state == ST_MAIN) {
        if (main_blink > 0) { … }  // 执行闪烁计数
        else { // 常亮/灭逻辑 }
        return;
    }
    // 参数模式闪烁
    uint32_t period = (state == ST_AMP ? 1000 : state == ST_CAB ? 600 : 350);
    if (now - led_timer >= period / 2) { led_timer = now; led_on ^= 1; … }
}
```

### 状态流转简图

```
主页 (ST_MAIN)
  ├─ Choose (PA1)  → 切换高亮效果链位置
  ├─ Enter  (PB0)  → 进入 ST_AMP / ST_CAB / ST_RVB
  ├─ Bypass (PB1)  → 切换高亮效果的旁通
  └─ 编码器        → 调节 g_master_volume

参数模式 (ST_AMP / ST_CAB / ST_RVB)
  ├─ 编码器        → 调节对应参数
  ├─ Bypass (PB1)  → 切换当前效果的旁通
  ├─ Enter  (PB0)  → 仅 CAB 切换 IR / RVB 切换子页（AMP 无动作）
  └─ Exit   (PA7)  → 保存参数并返回 ST_MAIN
```
